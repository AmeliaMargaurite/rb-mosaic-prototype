Using a style library

Below there are two sets of buttons. The first set uses the native HTML element 'button' and while the second set are created as Web Components which contain a 'button' element, and utilise the Shadow DOM.

Both sets of buttons use the same CSS file "buttons.css", and the same classes and styling. The native buttons receive their styling from the global styles, where the Web Component buttons import this file separately.

The way in which both types of buttons are used are very similar, as can be seen in the code snippet below each set. Which ever one is used, the visual design remains the same.

A benefit of using the Web Component using Shadow DOM option is the encapsulation of the code and styling. This safeguards the element from accidentally being affected by a change to a general CSS targeter which would normally affect them.
